WiFi Help
Networks available, how to connect, and the top fixes for common issues.
Available Networks
HD-Guest
No Password
For all visitors. One day of internet access per seven-day period. No password required — just agree to the splash screen terms.
HD-Members
Members Only
Password: hackerdojo
High-speed 1-gigabit connection for active members.
HD-Events
Event Attendees
Password provided by your event host. For use during ticketed or scheduled events.
Splash Screen
All Hacker Dojo WiFi networks require loading a splash screen. If it doesn't appear automatically after joining, open your browser and load any non-HTTPS URL, such as http://example.com.
Top 3 Fixes

These three steps resolve the vast majority of WiFi issues:

1
Disable VPN or Proxy
VPN and proxy configurations can block authentication with the access point. Turn them off before connecting, then re-enable after you've passed the splash screen.
2
Use Your @hackerdojo.com Username
Members: log in with your @hackerdojo.com username, not your Gmail or other email address. This is the most common mistake on the member network.
3
Verify the Event WiFi Password
Event organizers managing multiple calendar reservations: make sure the WiFi password you're using matches the current day's event, not a password from a different booking.
